Convolution Camera!

Play with the effects or change the values of the convolution matrix to see the results.
You can capture a frame and save it :)

This demo works best in Chrome, slower in Opera, and buggy in Firefox.
Everything else, either does not support getUserMedia or Web workers.

To change the number of web workers simply add it to the query string thus... If you want 2 web workers, .../camera/?2

Defaults to 10 workers, you can set 1 to 20. But you would get weird results if the number is not divisble by 240.
